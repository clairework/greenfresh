# 綠鮮生態農場電商網站
<img width="1887" height="923" alt="homepage" src="https://github.com/user-attachments/assets/e96b2e76-cb58-46c7-ac1b-a9ab8158c8d0" />



**Demo 網址**：https://clairework.github.io/greenfresh

圖片來源：  Unsplash、 Gemini

## 專案技術與特色
* 網站使用 React 建置而成
* 引入 React Router 與 Axios 等套件
* 引入 Bootstrap 5 樣式
* 開發後台管理系統
* 加入 ESLint Airbnb 風格
* 統一管理 API 路徑
* 透過 Prop-Types 與 Redux 傳遞跨元件資料
* 使用 Bootstrap Icons
* 使用 Bootstrap Toast 顯示通知訊息

## 前台
* 電商展示：首頁、關於我們、全部列表、商品介紹
* 購物車：刪除、修改商品數量
* 優惠券
* 結帳
  
## 後台
* 管理者登入
* 商品管理
* 優惠券管理
* 訂單管理

## 使用技術
* React
* React Hooks
* React Hook Form
* React Component
* React Router
* React Redux
* Axios
* Swiper
* Bootstrap 5
