## 專案技術與特色
* 網站使用 React 建置而成
* 引入 React Router 與 Axios 等套件
* 引入 Bootstrap 5 樣式
* 開發後台管理系統
* 加入 ESLint Airbnb 風格
* 統一管理 API 路徑
* 透過 Prop-Types 與 Redux 傳遞跨元件資料
* 使用 Bootstrap Icons
* 使用 Bootstrap Toast 顯示通知訊息
